# Scope Compliance Checklist

## In Scope
- Java desktop application using JavaFX
- Create a task
- View task list
- View task details
- Modify/edit a task
- Delete a task
- Mark task complete/incomplete
- Task fields:
  - Title (required)
  - Description (optional)
  - Due date (optional)
  - Priority (Low/Medium/High)
- Save and load tasks from a local file (JSON)

## Out of Scope
- Login/accounts
- Cloud sync
- Database server (SQL, Firebase, etc.)
- Notifications (text/email/push)
- Multi-user collaboration/sharing
- Mobile app
