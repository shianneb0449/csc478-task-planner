# Scope Statement
The personal task planner will be developed to run on Microsoft Windows 10 system. 

The software will function as a standalone desktop application running locally on a single computer. The application will not require network connection and will not include any server-side components. 

The personal task planner will not integrate with external systems or third party services. All task data will be stored locally by the application. 

The application will be intended for single user use only. 

The programming language used for this application will be Java. 

The personal task planner will use a graphical user interface to allow users to easily create, view, modify, and delete tasks. 

## In Scope
- Java desktop application using JavaFX
- Create a task
- View task list
- View task details
- Modify/edit a task
- Delete a task
- Mark task complete/incomplete
- Task fields:
  - Title (required)
  - Description (optional)
  - Due date (optional)
  - Priority (Low/Medium/High)
- Save and load tasks from a local file (JSON)

## Out of Scope
- Login/accounts
- Cloud sync
- Database server
- Notifications
- Multi user collaboration
- Mobile app
